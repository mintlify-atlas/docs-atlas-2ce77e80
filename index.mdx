<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold dark:text-gray-100 text-gray-900 mb-4">Quick Start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get ZipDrop running in minutes</p>
  
  <Steps>
    <Step title="Download and Install">
      Download the latest version from GitHub and move ZipDrop to your Applications folder. Launch the app and you'll see the ZipDrop icon in your menu bar.
      
      <CodeGroup>
      ```bash Homebrew
      brew install --cask zipdrop
      ```
      
      ```bash Manual Download
      # Download from GitHub releases
      # https://github.com/theforkproject-dev/zipdrop/releases
      ```
      </CodeGroup>
    </Step>
    
    <Step title="Try Demo Mode">
      ZipDrop starts in demo mode by default. Click the menu bar icon and drop a file to test it out. Files are saved locally to `~/Downloads/ZipDrop`.
      
      <Note>Demo mode is perfect for trying ZipDrop without cloud setup. Files stay on your Mac.</Note>
    </Step>
    
    <Step title="Configure Cloudflare R2 (Optional)">
      To upload files to the cloud, open Settings and enter your Cloudflare R2 credentials. Test your credentials, then save.
      
      ```json Example R2 Config
      {
        "account_id": "your-cloudflare-account-id",
        "bucket_name": "my-zipdrop-bucket",
        "access_key": "your-r2-access-key",
        "secret_key": "your-r2-secret-key",
        "public_url_base": "https://files.yourdomain.com"
      }
      ```
      
      <Tip>Need help with R2 setup? Check out our [Cloudflare R2 Setup Guide](/configuration/cloudflare-r2-setup)</Tip>
    </Step>
    
    <Step title="Upload Your First File">
      Click the ZipDrop icon, drag a file into the drop zone, and the public URL is instantly copied to your clipboard. Share it anywhere!
    </Step>
  </Steps>
</div>
